# Vehicle Registration & EV Adoption Analysis (India)

## Project Overview
This project analyzes Indian vehicle registration data to understand fuel trends, state-wise registrations, and EV adoption growth.

## Tools Used
- Python (Pandas, NumPy)
- Power BI
- Data Cleaning
- Exploratory Data Analysis
- Data Visualization

## Dataset
Source: Kaggle
- vehicle_registrations_500k.csv
- state_month_fuel_aggregated.csv

## Key Analysis
- Time-series registration trends
- State-wise vehicle comparison
- Fuel-type distribution analysis
- EV adoption pattern identification

## Data Challenges
- Handling missing values
- grossVehicleWeight = 0 treated as Not Applicable
- Performance optimization for large dataset

## Dashboard Features
- Interactive state filters
- Fuel-type comparison
- Time-based analysis visuals

## Future Scope
- Time-series forecasting
- Real-time data integration
